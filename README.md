# Build / Test / Release

## Setup Code Artifact

## Build the package.
```
hatch build
```

## Run tests
```
hatch run test
```

## Run linting
```
hatch run lint
```

## Run formating
```
hatch run fmt
```

## Run a tests for all supported Python versions.
```
hatch run all:test
```
